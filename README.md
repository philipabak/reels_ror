# README

This README would normally document whatever steps are necessary to get the
application up and running.

Project details:

* Ruby version: 2.7.2

* Rails version: > 7.0.8

* Database: sqlite3

Commands to run the project step by step:

* `bundle install`

* `rake db:create`

* `rake db:migrate`

* `rake db:seed`

* `rails s`

* `http://localhost:3000`
